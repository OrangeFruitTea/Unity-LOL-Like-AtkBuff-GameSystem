# 调试日志系统设计方案

## 一、需求分析

根据项目架构文档，基础设施层需要包含**日志调试**功能，具体要求：
- **多级别日志**：支持不同严重程度的日志记录
- **内嵌调试窗口**：在游戏内实时显示日志信息
- **高内聚、低耦合**：与其他系统解耦，易于扩展和维护
- **性能优化**：确保日志系统不影响游戏性能
- **支持未来扩展**：为热更新等功能预留接口

## 二、系统设计

### 2.1 架构定位

- **层级**：基础设施层
- **依赖关系**：仅依赖Unity引擎API，不依赖其他业务模块
- **设计模式**：Singleton单例模式 + 观察者模式

### 2.2 核心组件

```
LoggingSystem/
├── LogManager.cs          # 日志管理器（核心组件）
├── LogLevel.cs            # 日志级别枚举
├── ILogOutput.cs          # 日志输出接口
├── ConsoleOutput.cs       # 控制台输出实现
├── FileOutput.cs          # 文件输出实现
├── DebugWindow.cs         # 内嵌调试窗口
└── LogConfig.cs           # 日志配置
```

#### 2.3.3 LogManager 核心实现

```csharp
using System;
using System.Collections.Generic;
using UnityEngine;
using Basement.Utils;

namespace Basement.Logging
{
    public class LogManager : Singleton<LogManager>
    {
        private List<ILogOutput> _outputs = new List<ILogOutput>();
        private LogLevel _currentLevel = LogLevel.Debug;
        private bool _isInitialized = false;

        public void Initialize(LogConfig config)
        {
            if (_isInitialized)
                return;

            _currentLevel = config.DefaultLogLevel;
            
            // 添加默认输出
            _outputs.Add(new ConsoleOutput());
            _outputs.Add(new FileOutput(config.LogFilePath));
            _outputs.Add(DebugWindow.Instance);
            
            // 设置输出级别
            foreach (var output in _outputs)
            {
                output.SetLogLevel(_currentLevel);
            }
            
            _isInitialized = true;
            Log("日志系统初始化完成", LogLevel.Info, "LoggingSystem");
        }

        public void Log(string message, LogLevel level = LogLevel.Info, string tag = null)
        {
            if (level < _currentLevel)
                return;

            string logMessage = FormatLogMessage(message, level, tag);
            
            foreach (var output in _outputs)
            {
                output.Log(logMessage, level, tag);
            }
        }

        public void LogDebug(string message, string tag = null)
        { Log(message, LogLevel.Debug, tag); }

        public void LogInfo(string message, string tag = null)
        { Log(message, LogLevel.Info, tag); }

        public void LogWarning(string message, string tag = null)
        { Log(message, LogLevel.Warning, tag); }

        public void LogError(string message, string tag = null)
        { Log(message, LogLevel.Error, tag); }

        public void LogFatal(string message, string tag = null)
        { Log(message, LogLevel.Fatal, tag); }

        private string FormatLogMessage(string message, LogLevel level, string tag)
        {
            string time = DateTime.Now.ToString("HH:mm:ss.fff");
            string levelStr = level.ToString().ToUpper();
            string tagStr = string.IsNullOrEmpty(tag) ? "[GENERAL]" : $"[{tag}]";
            return $"[{time}] {levelStr} {tagStr} {message}";
        }

        // 动态添加输出目标
        public void AddOutput(ILogOutput output)
        {
            if (!_outputs.Contains(output))
            {
                output.SetLogLevel(_currentLevel);
                _outputs.Add(output);
            }
        }

        // 移除输出目标
        public void RemoveOutput(ILogOutput output)
        {
            if (_outputs.Contains(output))
            {
                _outputs.Remove(output);
            }
        }

        // 设置全局日志级别
        public void SetLogLevel(LogLevel level)
        {
            _currentLevel = level;
            foreach (var output in _outputs)
            {
                output.SetLogLevel(level);
            }
        }

        protected override void Awake()
        {
            base.Awake();
            // 使用默认配置初始化
            Initialize(new LogConfig());
        }
    }
}
```

#### 2.3.4 ConsoleOutput 实现

```csharp
using UnityEngine;

namespace Basement.Logging
{
    public class ConsoleOutput : ILogOutput
    {
        private LogLevel _logLevel = LogLevel.Debug;

        public void Log(string message, LogLevel level, string tag = null)
        {
            if (level < _logLevel)
                return;

            switch (level)
            {
                case LogLevel.Debug:
                    Debug.Log(message);
                    break;
                case LogLevel.Info:
                    Debug.Log(message);
                    break;
                case LogLevel.Warning:
                    Debug.LogWarning(message);
                    break;
                case LogLevel.Error:
                    Debug.LogError(message);
                    break;
                case LogLevel.Fatal:
                    Debug.LogError(message);
                    break;
            }
        }

        public void SetLogLevel(LogLevel level)
        {
            _logLevel = level;
        }
    }
}
```
