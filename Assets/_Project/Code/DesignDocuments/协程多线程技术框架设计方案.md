# 协程多线程技术框架设计方案

## 一、需求分析

根据项目架构文档，基础设施层需要支持高效的多线程处理能力，结合Unity引擎特点，设计一套基于协程的多线程技术框架，具体要求：
- **协程基础**：利用Unity协程处理轻量级异步任务
- **多线程支持**：结合真正的多线程处理CPU密集型任务
- **统一接口**：提供简洁易用的API，隐藏底层实现细节
- **线程安全**：确保多线程环境下的数据安全性
- **性能优化**：减少线程创建销毁开销，优化任务调度
- **与现有系统集成**：与资源管理、日志系统等基础设施层组件无缝集成

## 二、系统设计

### 2.1 架构定位

- **层级**：基础设施层
- **依赖关系**：仅依赖Unity引擎API和.NET标准库

#### 2.3.3 ThreadSafeQueue 线程安全队列

```csharp
namespace Basement.Threading
{
    public class ThreadSafeQueue<T>
    {
        private readonly Queue<T> _queue = new Queue<T>();
        private readonly object _lock = new object();
        private readonly SemaphoreSlim _semaphore = new SemaphoreSlim(0);

        /// <summary>
        /// 入队
        /// </summary>
        public void Enqueue(T item)
        {
            lock (_lock)
            {
                _queue.Enqueue(item);
            }
            _semaphore.Release();
        }

        /// <summary>
        /// 出队
        /// </summary>
        public bool TryDequeue(out T item)
        {
            if (_semaphore.Wait(0))
            {
                lock (_lock)
                {
                    item = _queue.Dequeue();
                    return true;
                }
            }
            item = default;
            return false;
        }

        /// <summary>
        /// 阻塞出队
        /// </summary>
        public T Dequeue()
        {
            _semaphore.Wait();
            lock (_lock)
            {
                return _queue.Dequeue();
            }
        }

        /// <summary>
        /// 尝试阻塞出队
        /// </summary>
        public bool TryDequeue(out T item, int timeoutMilliseconds)
        {
            if (_semaphore.Wait(timeoutMilliseconds))
            {
                lock (_lock)
                {
                    item = _queue.Dequeue();
                    return true;
                }
            }
            item = default;
            return false;
        }

        /// <summary>
        /// 获取队列长度
        /// </summary>
        public int Count
        {
            get
            {
                lock (_lock)
                {
                    return _queue.Count;
                }
            }
        }

        /// <summary>
        /// 清空队列
        /// </summary>
        public void Clear()
        {
            lock (_lock)
            {
                _queue.Clear();
            }
        }
    }
}
```

#### 2.3.4 CoroutineManager 协程管理器

```csharp
namespace Basement.Threading
{
    public class CoroutineManager : Singleton<CoroutineManager>
    {
        private readonly Dictionary<object, List<Coroutine>> _coroutines = new Dictionary<object, List<Coroutine>>();
        private readonly object _lock = new object();

        /// <summary>
        /// 启动协程
        /// </summary>
        public Coroutine StartCoroutine(IEnumerator enumerator, object owner = null)
        {
            Coroutine coroutine = base.StartCoroutine(enumerator);
            
            if (owner != null)
            {
                lock (_lock)
                {
                    if (!_coroutines.TryGetValue(owner, out var list))
                    {
                        list = new List<Coroutine>();
                        _coroutines[owner] = list;
                    }
                    list.Add(coroutine);
                }
            }
            
            return coroutine;
        }

        /// <summary>
        /// 停止协程
        /// </summary>
        public void StopCoroutine(Coroutine coroutine)
        {
            if (coroutine != null)
            {
                base.StopCoroutine(coroutine);
            }
        }

        /// <summary>
        /// 停止所有协程
        /// </summary>
        public void StopAllCoroutines(object owner = null)
        {
            if (owner != null)
            {
                lock (_lock)
                {
                    if (_coroutines.TryGetValue(owner, out var list))
                    {
                        foreach (var coroutine in list)
                        {
                            base.StopCoroutine(coroutine);
                        }
                        _coroutines.Remove(owner);
                    }
                }
            }
            else
            {
                base.StopAllCoroutines();
                lock (_lock)
                {
                    _coroutines.Clear();
                }
            }
        }

        /// <summary>
        /// 等待指定时间后执行
        /// </summary>
        public void WaitForSeconds(float seconds, Action onComplete, object owner = null)
        {
            StartCoroutine(WaitForSecondsCoroutine(seconds, onComplete), owner);
        }

        private IEnumerator WaitForSecondsCoroutine(float seconds, Action onComplete)
        {
            yield return new WaitForSeconds(seconds);
            onComplete?.Invoke();
        }

        /// <summary>
        /// 等待指定帧数后执行
        /// </summary>
        public void WaitForFrames(int frames, Action onComplete, object owner = null)
        {
            StartCoroutine(WaitForFramesCoroutine(frames, onComplete), owner);
        }

        private IEnumerator WaitForFramesCoroutine(int frames, Action onComplete)
        {
            for (int i = 0; i < frames; i++)
            {
                yield return null;
            }
            onComplete?.Invoke();
        }

        /// <summary>
        /// 等待直到条件满足
        /// </summary>
        public void WaitUntil(Func<bool> condition, Action onComplete, object owner = null)
        {
            StartCoroutine(WaitUntilCoroutine(condition, onComplete), owner);
        }

        private IEnumerator WaitUntilCoroutine(Func<bool> condition, Action onComplete)
        {
            yield return new WaitUntil(condition);
            onComplete?.Invoke();
        }
    }
}
```

#### 2.3.5 ThreadPool 自定义线程池

```csharp
namespace Basement.Threading
{
    public class ThreadPool
    {
        private readonly List<WorkerThread> _threads = new List<WorkerThread>();
        private readonly ThreadSafeQueue<ITask> _taskQueue = new ThreadSafeQueue<ITask>();
        private readonly object _lock = new object();
        private bool _isRunning = false;

        /// <summary>
        /// 初始化线程池
        /// </summary>
        public void Initialize(int minThreadCount = 2, int maxThreadCount = 8)
        {
            lock (_lock)
            {
                if (_isRunning)
                    return;

                _isRunning = true;
                
                // 创建线程
                for (int i = 0; i < minThreadCount; i++)
                {
                    CreateWorkerThread();
                }
            }
        }

        /// <summary>
        /// 创建工作线程
        /// </summary>
        private void CreateWorkerThread()
        {
            var thread = new WorkerThread(_taskQueue);
            _threads.Add(thread);
            thread.Start();
        }

        /// <summary>
        /// 提交任务
        /// </summary>
        public void SubmitTask(ITask task)
        {
            if (!_isRunning)
                throw new InvalidOperationException("ThreadPool is not running!");

            _taskQueue.Enqueue(task);
        }

        /// <summary>
        /// 停止线程池
        /// </summary>
        public void Shutdown()
        {
            lock (_lock)
            {
                if (!_isRunning)
                    return;

                _isRunning = false;
                
                // 停止所有线程
                foreach (var thread in _threads)
                {
                    thread.Stop();
                }
                
                _threads.Clear();
                _taskQueue.Clear();
            }
        }

        /// <summary>
        /// 获取当前任务数量
        /// </summary>
        public int TaskCount => _taskQueue.Count;

        /// <summary>
        /// 获取当前线程数量
        /// </summary>
        public int ThreadCount => _threads.Count;
    }

    /// <summary>
    /// 工作线程
    /// </summary>
    internal class WorkerThread
    {
        private readonly ThreadSafeQueue<ITask> _taskQueue;
        private Thread _thread;
        private bool _isRunning = false;

        public WorkerThread(ThreadSafeQueue<ITask> taskQueue)
        {
            _taskQueue = taskQueue;
        }

        public void Start()
        {
            _isRunning = true;
            _thread = new Thread(Run) { IsBackground = true, Name = "WorkerThread" };
            _thread.Start();
        }

        public void Stop()
        {
            _isRunning = false;
        }

        private void Run()
        {
            while (_isRunning)
            {
                try
                {
                    if (_taskQueue.TryDequeue(out ITask task, 100))
                    {
                        task.Execute();
                        task.OnComplete?.Invoke();
                    }
                }
                catch (Exception ex)
                {
                    Debug.LogError($"WorkerThread exception: {ex.Message}");
                }
            }
        }
    }
}
```

#### 2.3.6 TaskScheduler 任务调度器

```csharp
namespace Basement.Threading
{
    public class TaskScheduler : Singleton<TaskScheduler>
    {
        private ThreadPool _threadPool;
        private readonly Dictionary<TaskPriority, ThreadSafeQueue<ITask>> _taskQueues = new Dictionary<TaskPriority, ThreadSafeQueue<ITask>>();
        private bool _isInitialized = false;
        private readonly object _lock = new object();

        /// <summary>
        /// 初始化任务调度器
        /// </summary>
        public void Initialize(int minThreadCount = 2, int maxThreadCount = 8)
        {
            lock (_lock)
            {
                if (_isInitialized)
                    return;

                // 初始化任务队列
                foreach (TaskPriority priority in Enum.GetValues(typeof(TaskPriority)))
                {
                    _taskQueues[priority] = new ThreadSafeQueue<ITask>();
                }

                // 初始化线程池
                _threadPool = new ThreadPool();
                _threadPool.Initialize(minThreadCount, maxThreadCount);

                // 启动任务调度协程
                CoroutineManager.Instance.StartCoroutine(TaskDispatchCoroutine());

                _isInitialized = true;
            }
        }

        /// <summary>
        /// 任务调度协程
        /// </summary>
        private IEnumerator TaskDispatchCoroutine()
        {
            while (_isInitialized)
            {
                // 按优先级处理任务
                foreach (TaskPriority priority in Enum.GetValues(typeof(TaskPriority)))
                {
                    if (_taskQueues[priority].TryDequeue(out ITask task))
                    {
                        _threadPool.SubmitTask(task);
                        break; // 每次只处理一个任务，避免长时间阻塞主线程
                    }
                }
                yield return null;
            }
        }

        /// <summary>
        /// 提交任务到线程池
        /// </summary>
        public void SubmitTask(ITask task)
        {
            if (!_isInitialized)
                throw new InvalidOperationException("TaskScheduler is not initialized!");

            _taskQueues[task.Priority].Enqueue(task);
        }

        /// <summary>
        /// 提交Action作为任务
        /// </summary>
        public void SubmitTask(Action action, TaskPriority priority = TaskPriority.Normal, Action onComplete = null)
        {
            var task = new ActionTask(action, priority);
            task.OnComplete = onComplete;
            SubmitTask(task);
        }

        /// <summary>
        /// 启动协程
        /// </summary>
        public Coroutine StartCoroutine(IEnumerator enumerator, object owner = null)
        {
            return CoroutineManager.Instance.StartCoroutine(enumerator, owner);
        }

        /// <summary>
        /// 停止协程
        /// </summary>
        public void StopCoroutine(Coroutine coroutine)
        {
            CoroutineManager.Instance.StopCoroutine(coroutine);
        }

        /// <summary>
        /// 停止所有协程
        /// </summary>
        public void StopAllCoroutines(object owner = null)
        {
            CoroutineManager.Instance.StopAllCoroutines(owner);
        }

        /// <summary>
        /// 异步执行任务并返回结果
        /// </summary>
        public async Task<T> ExecuteAsync<T>(Func<T> function, TaskPriority priority = TaskPriority.Normal)
        {
            var taskCompletionSource = new TaskCompletionSource<T>();

            SubmitTask(() =>
            {
                try
                {
                    T result = function();
                    taskCompletionSource.SetResult(result);
                }
                catch (Exception ex)
                {
                    taskCompletionSource.SetException(ex);
                }
            }, priority);

            return await taskCompletionSource.Task;
        }

        /// <summary>
        /// 清理资源
        /// </summary>
        protected override void OnDestroy()
        {
            base.OnDestroy();
            
            if (_isInitialized)
            {
                _isInitialized = false;
                _threadPool?.Shutdown();
                CoroutineManager.Instance.StopAllCoroutines();
            }
        }

        protected override void Awake()
        {
            base.Awake();
            Initialize();
        }
    }

    /// <summary>
    /// Action任务
    /// </summary>
    internal class ActionTask : ITask
    {
        private readonly Action _action;

        public ActionTask(Action action, TaskPriority priority = TaskPriority.Normal)
        {
            _action = action;
            Priority = priority;
        }

        public TaskPriority Priority { get; }
        public Action OnComplete { get; set; }

        public void Execute()
        {
            _action?.Invoke();
        }
    }
}
```

## 三、使用示例

### 3.1 协程使用示例

```csharp
using Basement.Threading;
using UnityEngine;

public class CoroutineExample : MonoBehaviour
{
    private void Start()
    {
        // 启动协程
        TaskScheduler.Instance.StartCoroutine(MyCoroutine(), this);
        
        // 延迟执行
        TaskScheduler.Instance.StartCoroutine(DelayCoroutine(2.0f, () =>
        {
            Debug.Log("延迟2秒后执行！");
        }), this);
    }

    private IEnumerator MyCoroutine()
    {
        Debug.Log("协程开始");
        yield return new WaitForSeconds(1.0f);
        Debug.Log("等待1秒后继续");
        yield return new WaitForEndOfFrame();
        Debug.Log("等待帧结束后继续");
        
        // 嵌套协程
        yield return TaskScheduler.Instance.StartCoroutine(NestedCoroutine());
        
        Debug.Log("协程结束");
    }

    private IEnumerator NestedCoroutine()
    {
        Debug.Log("嵌套协程开始");
        yield return new WaitForSeconds(0.5f);
        Debug.Log("嵌套协程结束");
    }

    private IEnumerator DelayCoroutine(float seconds, Action onComplete)
    {
        yield return new WaitForSeconds(seconds);
        onComplete?.Invoke();
    }

    private void OnDestroy()
    {
        // 停止所有关联的协程
        TaskScheduler.Instance.StopAllCoroutines(this);
    }
}
```

### 3.2 多线程任务使用示例

```csharp
using Basement.Threading;
using UnityEngine;

public class ThreadingExample : MonoBehaviour
{
    private void Start()
    {
        // 提交普通任务
        TaskScheduler.Instance.SubmitTask(() =>
        {
            // 执行CPU密集型计算
            int result = CalculatePrimeNumbers(100000);
            Debug.Log($"计算完成，找到{result}个质数");
        }, TaskPriority.Normal);
        
        // 提交带完成回调的任务
        TaskScheduler.Instance.SubmitTask(() =>
        {
            // 执行耗时操作
            return GenerateLargeData();
        }, TaskPriority.High, (data) =>
        {
            // 在主线程中处理结果
            UpdateUIWithData(data);
        });
        
        // 使用async/await执行任务
        ExecuteAsyncTask();
    }

    private int CalculatePrimeNumbers(int max)
    {
        int count = 0;
        for (int i = 2; i <= max; i++)
        {
            if (IsPrime(i))
                count++;
        }
        return count;
    }

    private bool IsPrime(int number)
    {
        if (number <= 1) return false;
        if (number == 2) return true;
        if (number % 2 == 0) return false;

        var boundary = (int)Math.Floor(Math.Sqrt(number));

        for (int i = 3; i <= boundary; i += 2)
        {
            if (number % i == 0)
                return false;
        }

        return true;
    }

    private byte[] GenerateLargeData()
    {
        // 生成大量数据
        var data = new byte[1024 * 1024 * 10]; // 10MB
        new System.Random().NextBytes(data);
        return data;
    }

    private void UpdateUIWithData(byte[] data)
    {
        // 在主线程中更新UI
        Debug.Log($"数据生成完成，大小：{data.Length / 1024 / 1024}MB");
    }

    private async void ExecuteAsyncTask()
    {
        try
        {
            Debug.Log("异步任务开始");
            
            // 使用async/await执行任务
            int result = await TaskScheduler.Instance.ExecuteAsync(() =>
            {
                return CalculatePrimeNumbers(50000);
            });
            
            Debug.Log($"异步任务完成，找到{result}个质数");
        }
        catch (Exception ex)
        {
            Debug.LogError($"异步任务异常：{ex.Message}");
        }
    }
}
```

## 四、性能优化

1. **线程池大小优化**：根据硬件配置和项目需求调整线程池大小
2. **任务优先级**：根据任务重要性和紧急程度设置合适的优先级
3. **批量处理**：将多个小任务合并为一个大任务，减少线程切换开销
4. **避免频繁创建任务**：重用任务对象，减少GC压力
5. **异步/同步结合**：CPU密集型任务使用线程，I/O密集型任务使用协程
6. **资源限制**：避免同时执行过多资源密集型任务

## 五、线程安全注意事项

1. **UI更新**：只能在主线程中更新UI组件
2. **共享数据**：使用锁或其他同步机制保护共享数据
3. **Unity API**：大部分Unity API只能在主线程中调用
4. **资源访问**：避免在多线程中同时访问Unity资源
5. **事件回调**：确保事件回调在正确的线程中执行

## 六、扩展建议

1. **任务取消支持**：添加任务取消机制，允许取消正在执行的任务
2. **任务依赖**：支持任务之间的依赖关系，确保任务按顺序执行
3. **性能监控**：添加任务执行时间监控和统计功能
4. **内存优化**：添加对象池，重用任务对象
5. **多平台支持**：针对不同平台优化线程池配置
6. **与其他系统集成**：与资源管理、日志系统等进行更深入的集成

## 七、总结

本设计方案实现了一个基于Unity协程的多线程技术框架，具有以下特点：
- **统一接口**：提供简洁易用的API，隐藏底层实现细节
- **灵活调度**：结合协程和多线程处理不同类型的任务
- **线程安全**：确保多线程环境下的数据安全性
- **性能优化**：减少线程创建销毁开销，优化任务调度
- **易于扩展**：提供清晰的接口，方便添加新功能
- **与现有系统集成**：与资源管理、日志系统等基础设施层组件无缝集成

该框架能够有效提升游戏的性能和响应性，满足项目架构文档中对基础设施层的要求。
