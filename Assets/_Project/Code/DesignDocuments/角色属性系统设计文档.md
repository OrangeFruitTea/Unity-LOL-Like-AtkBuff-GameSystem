# 角色属性系统设计文档

## 1. 系统概述

角色属性系统是MOBA游戏的核心基础系统，负责管理游戏中所有可交互实体（英雄、小兵、野怪、防御塔等）的属性数据。该系统基于ECS（Entity Component System）架构，实现角色属性的动态管理与同步机制，为游戏中的各种交互提供数据支持。

### 2.2 ECS集成架构

角色属性系统基于ECS架构实现，通过以下组件和系统构建：

- **Entity**：游戏中的实体（英雄、小兵、野怪等）
- **Component**：实体的属性数据
- **System**：处理实体的属性更新逻辑

## 3. 核心组件设计

### 3.1 实体数据组件

```csharp
using System.Collections.Generic;
using Core.ECS;

namespace Core.Entity
{
    public struct EntityDataComponent : IEcsComponent
    {
        // 核心数据
        private Dictionary<EntityBaseDataCore, double> _coreData;
        // 额外数据
        private Dictionary<EntityBaseData, double> _bonusData;
        // 属性修饰符
        private Dictionary<EntityBaseDataCore, List<AttributeModifier>> _coreModifiers;
        private Dictionary<EntityBaseData, List<AttributeModifier>> _bonusModifiers;
        
        public void InitializeDefaults()
        {
            _coreData = new Dictionary<EntityBaseDataCore, double>
            {
                { EntityBaseDataCore.HpLimit, 1000 }, // 默认生命值上限1000
                { EntityBaseDataCore.CrtHp, 1000 }, // 默认当前生命值=上限
                { EntityBaseDataCore.MpLimit, 200 }, // 默认魔法值上限200
                { EntityBaseDataCore.CrtMp, 200 }, // 默认当前魔法值=上限
                { EntityBaseDataCore.AtkAD, 100 }, // 默认物理攻击100
                { EntityBaseDataCore.AtkAP, 50 }, // 默认魔法攻击50
                { EntityBaseDataCore.DefenceAD, 50 }, // 默认物理防御50
                { EntityBaseDataCore.DefenceAP, 30 }, // 默认魔法防御30
                { EntityBaseDataCore.AtkSpeed, 1.0 }, // 默认攻击速度1.0
                { EntityBaseDataCore.SkillCd, 1.0 }, // 默认技能冷却倍率1.0（无冷却缩减）
                { EntityBaseDataCore.CriticalRate, 0.05 }, // 默认暴击率5%
                { EntityBaseDataCore.MoveSpeed, 3.5 } // 默认移动速度3.5
            };
            _bonusData = new Dictionary<EntityBaseData, double>
            {
                { EntityBaseData.HpRecoverPerSecond, 5 }, // 默认每秒回血5
                { EntityBaseData.MpRecoverPerSecond, 2 }, // 默认每秒回蓝2
                { EntityBaseData.PenAD, 0 }, // 默认物理穿透0
                { EntityBaseData.PenAP, 0 }, // 默认魔法穿透0
                { EntityBaseData.LifeSteal, 0 }, // 默认生命偷取0
                { EntityBaseData.OmniVamp, 0 }, // 默认全能吸血0
                { EntityBaseData.AtkDistance, 1.5 }, // 默认攻击距离1.5
                { EntityBaseData.CriticalDamage, 1.5 }, // 默认暴击伤害150%
                { EntityBaseData.Resilience, 0 } // 默认韧性0
            };
            _coreModifiers = new Dictionary<EntityBaseDataCore, List<AttributeModifier>>();
            _bonusModifiers = new Dictionary<EntityBaseData, List<AttributeModifier>>();
        }

        public double GetData(EntityBaseDataCore data)
        {
            if (_coreData == null) return 0;
            double baseValue = _coreData.GetValueOrDefault(data);
            
            // 应用修饰符
            if (_coreModifiers.TryGetValue(data, out var modifiers))
            {
                foreach (var modifier in modifiers)
                {
                    if (modifier.IsValid())
                    {
                        baseValue = modifier.Apply(baseValue);
                    }
                }
            }
            
            return baseValue;
        }

        public double GetData(EntityBaseData data)
        {
            if (_bonusData == null) return 0;
            double baseValue = _bonusData.GetValueOrDefault(data);
            
            // 应用修饰符
            if (_bonusModifiers.TryGetValue(data, out var modifiers))
            {
                foreach (var modifier in modifiers)
                {
                    if (modifier.IsValid())
                    {
                        baseValue = modifier.Apply(baseValue);
                    }
                }
            }
            
            return baseValue;
        }

        public IReadOnlyDictionary<EntityBaseData, double> GetBonusData()
        {
            return _bonusData ?? (IReadOnlyDictionary<EntityBaseData, double>) new Dictionary<EntityBaseData, double>();
        }

        public IReadOnlyDictionary<EntityBaseDataCore, double> GetCoreData()
        {
            return _coreData ?? (IReadOnlyDictionary<EntityBaseDataCore, double>) new Dictionary<EntityBaseDataCore, double>();
        }

        public void SetData(EntityBaseDataCore data, double value)
        {
            _coreData ??= new Dictionary<EntityBaseDataCore, double>();
            _coreData[data] = value;
        }
        
        public void SetData(EntityBaseData data, double value)
        {
            _bonusData ??= new Dictionary<EntityBaseData, double>();
            _bonusData[data] = value;
        }
        
        public void AddModifier(EntityBaseDataCore data, AttributeModifier modifier)
        {
            _coreModifiers ??= new Dictionary<EntityBaseDataCore, List<AttributeModifier>>();
            if (!_coreModifiers.TryGetValue(data, out var list))
            {
                list = new List<AttributeModifier>();
                _coreModifiers[data] = list;
            }
            list.Add(modifier);
        }
        
        public void AddModifier(EntityBaseData data, AttributeModifier modifier)
        {
            _bonusModifiers ??= new Dictionary<EntityBaseData, List<AttributeModifier>>();
            if (!_bonusModifiers.TryGetValue(data, out var list))
            {
                list = new List<AttributeModifier>();
                _bonusModifiers[data] = list;
            }
            list.Add(modifier);
        }
        
        public void RemoveModifier(EntityBaseDataCore data, string modifierId)
        {
            if (_coreModifiers.TryGetValue(data, out var list))
            {
                list.RemoveAll(m => m.Id == modifierId);
            }
        }
        
        public void RemoveModifier(EntityBaseData data, string modifierId)
        {
            if (_bonusModifiers.TryGetValue(data, out var list))
            {
                list.RemoveAll(m => m.Id == modifierId);
            }
        }
        
        public void Update(float deltaTime)
        {
            // 清理过期的修饰符
            if (_coreModifiers != null)
            {
                foreach (var kvp in _coreModifiers)
                {
                    kvp.Value.RemoveAll(m => !m.IsValid());
                    foreach (var modifier in kvp.Value)
                    {
                        modifier.Update(deltaTime);
                    }
                }
            }
            
            if (_bonusModifiers != null)
            {
                foreach (var kvp in _bonusModifiers)
                {
                    kvp.Value.RemoveAll(m => !m.IsValid());
                    foreach (var modifier in kvp.Value)
                    {
                        modifier.Update(deltaTime);
                    }
                }
            }
        }
    }
}
```

### 3.2 属性修饰符

```csharp
namespace Core.Entity
{
    public class AttributeModifier
    {
        public string Id { get; set; }
        public double Value { get; set; }
        public ModifierType Type { get; set; }
        public float Duration { get; set; }
        public float ElapsedTime { get; set; }
        public int Priority { get; set; }
        
        /// <summary>
        /// 检查modifier是否有效
        /// </summary>
        public bool IsValid()
        {
            return Duration <= 0 || ElapsedTime < Duration;
        }
        
        /// <summary>
        /// 更新modifier
        /// </summary>
        public void Update(float deltaTime)
        {
            if (Duration > 0)
            {
                ElapsedTime += deltaTime;
            }
        }
        
        /// <summary>
        /// 应用修饰符到基础值
        /// </summary>
        public double Apply(double baseValue)
        {
            switch (Type)
            {
                case ModifierType.Flat:
                    return baseValue + Value;
                case ModifierType.Percentage:
                    return baseValue * (1 + Value);
                default:
                    return baseValue;
            }
        }
    }
    
    public enum ModifierType
    {
        Flat,       // 固定值
        Percentage  // 百分比
    }
}
```

### 3.3 属性计算系统

```csharp
using Core.ECS;
using Core.Entity;

namespace Core.Systems
{
    public class AttributeCalculationSystem : IEcsSystem
    {
        public void Update(float deltaTime, EcsWorld world)
        {
            // 遍历所有具有EntityDataComponent的实体
            foreach (var entity in world.GetEntitiesWithComponent<EntityDataComponent>())
            {
                var dataComponent = entity.GetComponent<EntityDataComponent>();
                dataComponent.Update(deltaTime);
                entity.SetComponent(dataComponent);
            }
        }
    }
}
```

2. **修饰符应用**：按优先级应用所有有效的属性修饰符

### 6.2 等级成长系统

```csharp
public void ApplyLevelGrowth(EcsEntity entity, int level)
{
    var dataComponent = entity.GetComponent<EntityDataComponent>();
    var config = GetCharacterConfig(entity);
    
    foreach (var growth in config.LevelGrowth)
    {
        double baseValue = dataComponent.GetData(growth.Key);
        double growthValue = growth.Value * (level - 1);
        dataComponent.SetData(growth.Key, baseValue + growthValue);
    }
    
    entity.SetComponent(dataComponent);
}
```

## 8. 性能优化策略

### 8.1 属性计算优化
- **缓存计算结果**：缓存频繁访问的属性值
- **批量更新**：批量处理属性变更
- **修饰符管理**：使用对象池管理属性修饰符

### 8.2 内存优化
- **字典优化**：使用预分配容量的字典
- **组件复用**：复用ECS组件实例
- **垃圾回收**：减少临时对象创建

### 8.3 计算优化
- **优先级排序**：按优先级处理修饰符
- **无效修饰符清理**：定期清理过期修饰符
- **计算策略**：根据属性类型选择不同的计算策略

## 10. 结论

本设计文档基于ECS架构，实现了一个灵活、高效的角色属性系统，支持MOBA游戏的各种需求。系统采用模块化设计，提供了丰富的扩展点，便于后续功能的添加和修改。同时，系统预留了网络同步接口，为未来的多人游戏功能做好了准备。

通过合理的性能优化策略，系统能够在大量实体同时存在的情况下保持良好的性能表现。与其他系统的集成设计也确保了系统的可扩展性和可维护性。

本设计文档为角色属性系统的实现提供了详细的指导，包括系统架构、核心组件、接口定义、实现细节等方面，为开发团队提供了清晰的开发路线图。
